D** Project (insert ../crumbs.html here) # CAMLG4 CAMLG4 is an Objective Caml module that provides a BigArray like interface in order to manipulate large amount of data using the Altivec SIMD extension on Mac G4 and G5. This archived version has been running on a Cube G4 with Linux Yellow Dog 2.4.18, Objective CAML 3.06 and has been compiled with g++ 2.95.2. CAMLG4 has been tested on PowerPC G4, G5 and on XServe G5. !!! WARNING This C++ library was designed for early Macintosh Hardware (Cube G4 and early G5). It is archived as a research artifact. If you're into software genealogy, this is the great ancestor of [EVE](eve.html). ## Download You can download [CAMLG4 1.0 here](../media/camlg4.tar.gz) ## Bibliography [**CamlG4: une bibliothèque de calcul parallèle pour Objective Caml.** by *J Falcou* and *J Sérot*, in JFLA 2003 (p. 139-152)](../paper/camlg4.pdf) ----------------------------------------------------------------------------------------------------
